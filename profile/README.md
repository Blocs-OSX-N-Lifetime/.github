## Overview

Blocs for macOS is a specialized web design tool solution designed for users who want a reliable, OSX-native tool with clear controls and stable performance.

The native integration works seamlessly with macOS services and features. Launch through Spotlight, preview with Quick Look, or access through Services menu.

---

## Key Features

- Native macOS application
- Universal binary for all Macs
- Dark mode support
- Quick Look integration
- Spotlight indexing

## Additional Information

Blocs OSX delivers capabilities specifically tuned for the platform's strengths. The design respects established conventions while introducing refinements that enhance productivity.
